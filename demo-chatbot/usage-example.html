Medcor AI Chat Widget

Kolayca entegre edilebilen, HeyGen avatar olmadan çalışan tıbbi asistan chatbot

🚀 Canlı Demo

Sayfanın sağ alt köşesinde chat widget'ını görebilirsiniz. Tıklayarak test edebilirsiniz!

Özellikler:
  • Türkçe ve İngilizce destek
  • Ses tanıma (mikrofon)
  • Tıbbi sorular için akıllı yanıtlar
  • Randevu alma yönlendirmesi
  • Doktor bilgileri
  • Çalışma saatleri ve konum bilgisi
📝 Script Tag ile Entegrasyon

En kolay entegrasyon yöntemi. Sadece bu script'i sayfanıza ekleyin:

<script src="https://your-replit-url.replit.app/api/chat-widget.js"></script>

Widget otomatik olarak sayfanızın sağ alt köşesinde görünecektir.

⚛️ React Component Olarak

React projelerinizde component olarak kullanabilirsiniz:

import { MedcorChatWidget } from './medcor-chat-widget'; function App() { return ( <div> {/* Your app content */} <MedcorChatWidget /> </div> ); }
🔧 Manuel Başlatma

Programatik olarak kontrol etmek isteyenler için:

// Widget'ı manuel olarak başlat const chatWidget = new MedcorChatWidget({ apiUrl: 'https://your-api.com', language: 'tr', position: 'bottom-right' }); // Widget'ı aç/kapat chatWidget.open(); chatWidget.close();
🎨 Özelleştirme Seçenekleri

Widget'ı kendi ihtiyaçlarınıza göre özelleştirebilirsiniz:

const widget = new MedcorChatWidget({ apiUrl: 'https://api.medcor.com', language: 'tr', // 'tr' veya 'en' position: 'bottom-right', theme: 'purple', autoOpen: false, welcomeMessage: 'Özel karşılama mesajı' });
